# 🎮 Color Switch+ (Web Game)

A modern, minimal, and addictive **Color Switch-style web game** built using **HTML, CSS, and JavaScript (Canvas API)**.  
Control a bouncing ball, match colors, avoid obstacles, and survive as long as possible!

---

## 🚀 Features

- 🎯 Simple one-tap gameplay  
- 🎨 Dynamic color switching (auto + random)  
- ⚡ Smooth physics-based movement  
- 🧩 Rotating and mixed obstacles  
- 🌈 Power-ups (optional extension)  
- 🌙 Minimal neon UI design  
- 📱 Responsive (works on desktop & mobile)  

---

## 🕹️ Gameplay

- Tap / Click to make the ball jump  
- Pass through obstacles of the **same color**  
- Hitting a **different color = Game Over**  
- Ball color changes automatically over time  
- Score increases as you progress

## ⚙️ Game Logic

- Gravity-based ball movement  
- Tap input for jump  
- Continuous loop using `requestAnimationFrame`  
- Collision detection based on color  
- Random color switching every few seconds
